Variables y la función print()

Una variable es un nombre que tú escoges, al que le asignas un valor. El signo = es el que hace la asignación. Es como decir "ponle este nombre a este dato".

titulo = "Crónica de los archivos perdidos" autor = "B. del Castillo" año = 1952

Tres tipos importantes de datos:

  • Los strings (textos) van entre comillas: "hola" o 'hola'.
  • Los enteros (números sin coma) van sin comillas: 1952.
  • Los decimales (con coma) usan punto en vez de coma: 3.14.

Para mostrar el contenido de una variable en pantalla, se usa la función print(). La función imprime entre paréntesis lo que le pongas. Y para pegar texto con texto, se usa el signo +:

print(titulo) print("Autor: " + autor) print("Año: " + str(año))

Detalle importante: el último print usa str(año) porque año es un número y queremos pegarlo con texto. Para pegar números con texto, hay que convertir el número a texto primero. Eso lo hace str().

Índices en cadenas de texto

Cuando tienes un string (un texto), cada letra ocupa una posición. La primera letra está en la posición 0, no en la 1. La segunda en la 1. La tercera en la 2. Y así sucesivamente.

palabra = "BIBLIOTECA" print(palabra[0]) # B print(palabra[3]) # L print(palabra[9]) # A

Si quieres saber cuántas letras tiene una palabra, usas la función len():

print(len(palabra)) # 10

Cuidado: la palabra BIBLIOTECA tiene 10 letras, pero la última está en la posición 9 (no en la 10), porque las posiciones empiezan en 0. Si tratas de pedir palabra[10], Python se queja porque ahí no hay nada.

Y para pegar letras y formar una nueva palabra, simplemente las sumas con +:

letra1 = palabra[2] letra2 = palabra[5] resultado = letra1 + letra2 print(resultado) # BO

Listas y bucles for

Una lista es una secuencia de elementos. Se crea con corchetes y comas:

letras = ["L", "A", "L", "L", "A", "V", "E"]

Igual que con los strings, tiene índices que empiezan en 0. Y puedes ver cuántos elementos tiene con len():

print(letras[0]) # L print(letras[3]) # L print(len(letras)) # 7

Lo más útil de una lista es que puedes recorrerla entera con un bucle for. Lo que va después del in es la lista. La variable después del for va tomando los valores uno por uno:

for letra in letras: print(letra) # Imprime cada letra en su propia línea: L, A, L, L, A, V, E

Y si quieres juntar todas las letras en una sola palabra, vas pegándolas una a una en una variable que empieza vacía:

palabra = "" # empieza vacía for letra in letras: palabra = palabra + letra print(palabra) # LALLAVE

Ese patrón — empezar con algo vacío y acumular dentro de un bucle — es uno de los más importantes en programación. Lo vas a usar mucho.

Condicionales: if, elif, else

Una condicional es código que solo se ejecuta si se cumple cierta condición. La estructura básica:

if condicion: # esto se ejecuta SOLO si condicion es verdadera else: # esto se ejecuta SOLO si condicion es falsa

Las condiciones se hacen con comparaciones:

  • == es "igual a" (ojo: doble igual; uno solo es asignación)
  • != es "diferente de"
  • <, >, <=, >= para números

Cuando tienes más de dos opciones, se usa elif (abreviación de "else if") entre el primer if y el último else:

año = 1962 if año == 1952: pista = "al norte" elif año == 1962: pista = "al sur" elif año > 1962: pista = "al este" else: pista = "está en el libro" print(pista) # al sur

Python evalúa en orden: revisa el primer if, si es falso pasa al elif, y así sucesivamente. En cuanto encuentra uno verdadero, ejecuta esa rama y se sale. Los demás elif ya no los revisa.

Importante: las líneas DENTRO de cada rama van indentadas (con 4 espacios). Si no, Python no entiende que pertenecen al if.

Desempaque en for + condicional con and

Las listas pueden tener elementos compuestos. Por ejemplo, una lista de parejas donde cada elemento tiene dos datos:

habitaciones = [ ("ARCHIVO", "punto"), ("ESCRITORIO", "estrella"), ("SALÓN", "cruz"), ]

Esas parejas se llaman tuplas. Cuando recorres una lista de tuplas con un for, puedes desempacar los dos valores en dos variables al mismo tiempo:

for nombre, marca in habitaciones: print(nombre + " tiene marca " + marca)

Y cuando una condición depende de dos cosas al mismo tiempo, se usa la palabra and (que significa "y"):

if marca == "estrella" and nombre[0] == "E": print("¡Esta es la habitación!")

La condición se cumple SOLO si las DOS cosas son verdaderas. Si una sola falla, la condición entera es falsa.

Otro detalle útil: nombre[0] te da la primera letra del nombre (acuérdate, posición 0). Es exactamente lo que vimos en la pista 2.

Adentro encuentran cajas de archivos nunca catalogados, correspondencia de los años cuarenta, fotografías en blanco y negro de un colegio que ya no existe como era entonces, y un cuaderno de notas con la letra inconfundible de B. del Castillo. El primer párrafo dice:

Si encontraste esta habitación, es porque aprendiste a leer más que palabras. Los números también hablan. Las listas guardan secretos. Y el código — cualquier código — es solo una forma de decir cosas que no todos deben entender todavía.

"¿Ves? Eso es exactamente lo que aprendiste hoy. Variables para guardar datos. Strings con índices para sacar letras. Listas para agrupar cosas. Bucles para recorrerlas. Y condicionales para tomar decisiones. No son solo herramientas de programación — son una forma de pensar. Blas del Castillo lo sabía desde 1952."

"En cuarenta años nunca pensé que ese cuarto se volvería a abrir. Gracias. Y no olviden que el cuaderno de B. del Castillo tiene más páginas. Hay más cosas guardadas. El Club de la Cifra tiene trabajo pendiente."

Caso cerrado — la llave encontrada.

Un libro sin catalogar. Once páginas con agujas. Setenta y cuatro años de silencio. Y un cuarto con archivos que nadie había visto desde los años sesenta. Todo descifrado con cinco conceptos de Python y mucho pensamiento lógico.

El cuaderno de Blas del Castillo tiene más páginas. El archivo tiene más cajas. Y hay un nombre en las fotografías que Sara no reconoce — aparece en veinte fotos distintas, siempre de espaldas. Nunca de frente. ¿Quién es?

Lo que aprendiste en este caso

PISTA I
Variables y print()

Guardar datos con nombre. Imprimir resultados. Pegar texto con +.

PISTA II
Índices de string

Acceder a letras por posición (desde 0). len() para contar caracteres.

PISTA III
Listas y bucle for

Agrupar elementos. Recorrerlos con for ... in. Acumular en variable vacía.

PISTA IV
Condicionales if/elif/else

Tomar decisiones según condiciones. Operadores de comparación == != < >.

PISTA V
Desempaque + and

Tuplas en listas. Desempacar con for a, b in lista. Dos condiciones con and.
